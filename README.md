## Overview

**Know Sudurpaschim Well** is a comprehensive informational website designed to provide users with detailed insights into Sudurpaschim Province — Nepal's westernmost province. The platform serves locals, students, tourists, and researchers by presenting curated, accessible, and engaging content about the region's culture, geography, history, and ongoing development.

## 🎯 Objectives

- **Promote Awareness** — Educate the public about Sudurpaschim's identity, significance, and people.
- **Cultural Heritage** — Showcase traditions, festivals (Gaura, Deuda), music, and ethnic diversity.
- **Geographical Information** — Present mountains, rivers, terrain, and climate of the far west.
- **Historical Records** — Document ancient kingdoms, regional history, and provincial milestones.
- **Tourism Promotion** — Highlight natural wonders like Api Nampa, Khaptad National Park, and Shuklaphanta.
- **Development Updates** — Track infrastructure, economy, and provincial governance progress.

---

## 📚 Content Areas

### 🏔️ Geography
Mountains, rivers, terrain types, climate zones, and natural landmarks of the province.

### 🎭 Culture & Heritage
Festivals like Gaura Parva, Deuda folk music, traditional attire, local cuisine, and ethnic communities including Tharu, Kami, and Chhetri.

### 📜 History
Ancient trade routes, the Doti Kingdom, unification with Nepal, and key historical events shaping the province.

### 🧭 Tourism
- **Api Nampa Conservation Area** — pristine Himalayan wilderness
- **Khaptad National Park** — highland meadows and spiritual significance
- **Shuklaphanta National Park** — one-horned rhino and wildlife habitat
- **Atariya, Dadeldhura, Baitadi** — cultural and heritage sites

### 🏗️ Development
Roads, hydropower, health infrastructure, education, and economic growth in Sudurpaschim.

## 🚀 Getting Started

### Prerequisites

- Node.js v18 or higher
- npm v9 or higher

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/know-sudurpaschim.git

# Navigate to the project
cd know-sudurpaschim

# Install dependencies
npm install

# Start the development server
npm start
```

The app will run at **http://localhost:3000**
